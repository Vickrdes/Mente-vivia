# 🧠 Mente Viva - mHealth Solution

O **Mente Viva** é um aplicativo focado no auxílio a idosos com predisposição ao Alzheimer ou em estágio inicial (nível 1), bem como seus familiares e cuidadores. Desenvolvido durante a disciplina de **Metodologia Ágil**, o projeto une tecnologia e impacto social.

---

## 🎯 Objetivos do Projeto
* **Estimulação Cognitiva:** Atividades interativas para exercitar a mente.
* **Organização de Rotina:** Calendário e tarefas com lembretes automáticos.
* **Apoio ao Cuidador:** Conteúdos informativos e monitoramento de bem-estar.

## ✨ Funcionalidades Principais
- ✅ **Gestão de Tarefas:** Lista de afazeres e calendário integrado.
- 📚 **Biblioteca de Conteúdo:** Artigos sobre Alzheimer e receitas saudáveis.
- 🎮 **Jogos Cognitivos:** Jogo da memória, sequência numérica e desafios lógicos (Genius).
- 🔔 **Notificações:** Sistema de lembretes para compromissos e medicações.

## 🛠️ Tecnologias Utilizadas
* **Frontend:** FlutterFlow / Dart
* **Backend & Database:** Firebase (Firestore)
* **Gestão:** Trello (Kanban), Scrum (Sprints semanais)

## 📂 Estrutura do Repositório
* `lib/`: Contém todas as telas (.dart) desenvolvidas para o aplicativo.
* `database/`: Regras de segurança e índices do Firestore.
* `pubspec.yaml`: Arquivo de configuração de dependências do Flutter.
